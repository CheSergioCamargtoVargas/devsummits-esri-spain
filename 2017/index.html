Inscríbete Developer Summit Esri España

Qué te interesa

Te informamos de que tus datos de carácter personal se incorporarán a los ficheros de ESRI ESPAÑA SOLUCIONES GEOESPACIALES, S.L. y en los servidores de Mailchimp en Estados Unidos con la única finalidad de realizar comunicaciones relacionadas con el evento "Developer Summit" al que te inscribes, en esta o en futuras ediciones y nunca serán cedidos a terceros salvo expresa solicitud del usuario.

Es posible que en el desarrollo del evento se realicen fotografías y/o vídeos, los cuales podrán ser publicados en nuestras páginas Webs www.esri.es, desarrolladores.esri.es y en nuestros perfiles en las Redes Sociales. Si no deseas que tu imagen y/o voz sean tratados con esta finalidad, por favor, no participes en los posados fotográficos y audiovisuales que se realicen durante el evento.

Por último, informarte de que en caso de que nos facilites datos de terceros, incluidas personas de contacto, deberás obtener su consentimiento para ello e informarles de todo lo establecido en esta cláusula.

Con las limitaciones establecidas en la Ley, podrán ejercerse los derechos de acceso, rectificación, cancelación y oposición mediante escrito, al que se adjunte copia del DNI del interesado, dirigiéndose por escrito a C/ Emilio Muñoz, 35, 28037 – Madrid.

AGENDA

TODA LA PROGRAMACIÓN DEL EVENTO MIÉRCOLES 28 - JUNIO

Hora Descripción
9:00 - 9:15 Acreditaciones
Recogida de las acreditaciones de los asistentes
9:15 - 9:45 Iniciativas para desarrolladores
En esta primera sesión hablaremos de las iniciativas de Esri España para desarrolladores como: Awesome ArcGIS, ArcGIS Samples, ...
9:45 - 11:15 Sesión inaugural
En una intensa sesión haremos un repaso rápido por las últimas novedades: Arcade, layer views, smartmapping, API JS 4.3, Runtime v100, rendimiento, 3D, nuevos vector tiles, 3D , Python API, Chef, Web AppBuilder, etc.
11:15 - 11:45 Descanso: café - networking
11:45 - 12:00 Programa Experto Esri España
Este es un programa de formación dirigido a desarrolladores que quieren ampliar sus conocimientos en programación sobre la Plataforma ArcGIS. Explicaremos cómo va a ser, cómo podéis apuntaros y experiencias de la primera edición.
12:00 - 12:45 Arquitecturas híbridas
Veremos un caso de uso de una arquitectura híbrida de usuarios que nos permita reaprovechar y extender la plataforma ArcGIS combinando usuarios nominales con un sistema de usuarios propio.
12:45 - 13:15 GeoInformation model
ArcGIS Online y Portal for ArcGIS soportan 79 tipos de elementos (KML, CSV, ZIP, GeoJSON, LYR, MMPK, MXD, WFS, WMS, Feature Service, Image Service, etc.), algunos de ellos alojados y otros referenciados en sistemas de terceros. Destriparemos el modelo para aprender a sacarle el máximo partido.
13:15 - 13:45 Licenciamiento para desarrolladores
Explicaremos cómo funciona el nuevo sistema de licenciamiento para que los desarrolladores puedan adquirir todas las tecnologías para sus desarrollo y el licenciamiento para poder poner en producción cualquier desarrollo.
13:45 - 15:00 Descanso
15:00 - 17:30

Talleres

ArcGIS REST API - Sala 1
Haremos un repaso a las APIs REST disponibles en ArcGIS y aprenderemos a crear un servicio, añadirle capas, manipular sus datos, pero también para buscar y gestionar items dentro de una organización usando Node.js. 		ES6, patrones de desarrollo y ArcGIS JavaScript API 4 - Sala 2
JavaScript no deja de avanzar y el API de ArcGIS para JavaScript avanza de igual modo. Veremos cómo se ha implementado la versión 4 del API y cómo podemos aprovechar las nuevas features de ES6 a la hora de trabajar con ella..
Aplicaciones configurables - Sala 3
Veremos cómo crear aplicaciones configurables como los storymaps, geoform, etc. Para ello: 1) Desarrollaremos una desde cero, 2) Modificaremos una existente y 3) Veremos usar Web AppBuilder para crear una. 		Web Performance - Sala 4
Aprenderemos cómo optimizar el rendimiento de lado del cliente. Para ello utilizaremos herramientas y trucos que nos permitirán evaluar y reducir el tiempo de carga de nuestras aplicaciones web. Veremos herramientas como: Webpack, Node.js, Grunt, etc.
